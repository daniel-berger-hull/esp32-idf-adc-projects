HTTPServer_ADCdisplay
--------------------

  Simple HTTP server that serve an index.html page on which you can see the ADC reading done on pin GPIO34
  (ADC1 :: ADC_CHANNEL_6)

  Status: The page is flicking as the http refresh rate is very low
